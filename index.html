USER GUIDE

In length, weight, and temperature conversion. First select type of length, weight, and temperature. Then enter the value of the selected type. The output will be shown other types of conversion values.

In area and volume conversion. Enter foot and Inchie value of the width, length, and height properly. The output of conversion will be shown after clicking on the button.
Example:
if width: 5 foot 6 Inchie , length: 10 foot and height: 11 Inchie. In this case, The width menu input will be foot value 5 and Inchie value 6. Length menu input will be foot value 10 and Inchie value 0 or empty. Height menu input will be foot value 0 or empty and Inchie value 11.
